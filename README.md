# Attendance Register System
A simple **Java console application** to manage worker attendance.  
This program allows workers to sign attendance, view attendance records, and exit the system.

---

## Features
- Sign attendance with **worker name and ID**
- Display all attendance records with timestamps
- Exit the program gracefully
- Demonstrates **Object-Oriented Programming (OOP)** concepts: classes, objects, encapsulation, and collections

---

## Tech Stack
- **Language:** Java  
- **Core Concepts:** OOP, Collections (List), LocalDateTime, Scanner
